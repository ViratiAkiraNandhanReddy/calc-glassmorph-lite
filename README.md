# 🌿 Glassmorphism Calculator

A modern, premium glassmorphism calculator with a beautiful nature background, smooth UI, and calculation history.  
Designed with performance, accessibility, and clean aesthetics in mind, this project demonstrates a stylish web calculator built using pure HTML, CSS, and JavaScript.

---

## ✨ Overview

Glassmorphism Calculator is a lightweight, responsive, and visually elegant calculator interface inspired by modern glass UI design trends.  
It provides essential arithmetic operations, persistent history, keyboard input support, and a calming full-screen nature backdrop to enhance the user experience.

This project is ideal for learning front-end UI design, DOM manipulation, and local storage usage in a real-world styled application.

---

## 🎯 Features

- Premium glassmorphism interface
- Nature-themed full-screen background
- Basic arithmetic operations
- Persistent calculation history
- Keyboard input support
- Responsive layout for mobile and desktop
- Smooth hover and interaction animations
- Clean typography with modern styling
- Local storage history saving
- Lightweight and dependency-free
- SEO-optimized structure
- Accessible semantic HTML layout

---

## 🧮 Supported Operations

The calculator supports common arithmetic calculations:

- Addition
- Subtraction
- Multiplication
- Division
- Decimal input
- Continuous operations
- Clear and delete functions

It also prevents invalid operations such as division by zero.

---

## 🕘 History System

The calculator stores recent calculations locally in the browser so that users can revisit past results even after refreshing the page.

History features include:

- Automatic saving of calculations
- Persistent storage in browser
- Quick result reuse by clicking history items
- History clearing option
- Limited to recent entries for performance

---

## ⌨️ Keyboard Support

The calculator can be fully operated using the keyboard.

Supported inputs:

- Number keys for digits
- Decimal point
- Arithmetic operators
- Enter for calculation
- Backspace for delete
- Escape for clear

This enables fast and accessible interaction without relying solely on mouse or touch.
